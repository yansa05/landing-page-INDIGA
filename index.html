ABOUT US

Indiga Studio

Indiga is an Indonesia Game Studio , independent game development and publishing studio. indiga was made in 2019, the first game released was jowo-the next leader but unfortunately the game failed or no one was downloaded. 2020 indiga tries to release a horror game, namely Hantu Survival. Until now, Hantu Survival has received 40000 downloads. now indiga is developing game Hantu Survival 2 and many other mobile games. Let's Have Fun

Founder

Yansa really likes games. While still in college, he learned to make games and tried to build a PC game with the title Hantu Survival. now he is trying to set up an indiga studio with mobile game products such as Hantu Survival, free shoot and others.
